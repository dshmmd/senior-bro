# 2026-06-11 — v0.1 foundation shipped (P1–P8)

## What exists and works

- **Monorepo**: npm workspaces (`server`, `web`). Verification gate:
  `npm run typecheck && npm run build && npm run smoke` — all green at ship time.
- **Server** (Hono, port 4747): config in `~/.senior-bro/config.json`, data in
  `~/.senior-bro/data.db` via built-in `node:sqlite` (no native deps).
  Providers: Anthropic via official SDK (default `claude-opus-4-8`, prompt
  caching on system block), OpenAI via raw fetch. Key is validated with a tiny
  round-trip on save.
- **Interview engine** (`server/src/prompts.ts`): phased system prompt
  (warmup → behavioral → technical → design → wrap), `[INTERVIEW_COMPLETE]`
  end token, voice mode constrains replies to <80 words / no markdown for TTS.
  Evaluation is a separate strict-JSON call (`extractJson` handles fenced/prose
  wrapping). Weaknesses persist and are injected into future interview prompts.
- **Calibration**: 5 generated questions → graded → level stored on profile.
- **Coaching**: drill session targeting one weakness, weakness status lifecycle
  open → improving → resolved.
- **Web**: React+Vite SPA, state-machine routing in `App.tsx`
  (setup → profile → calibration → dashboard → interview). Voice via Web Speech
  API (`web/src/voice.ts`): continuous recognition with auto-restart on silence,
  barge-in (mic tap cancels TTS).
- **Skill packs**: `skills/{google,amazon,meta,startup}.md` — frontmatter
  (company/roles/summary) + playbook body injected into the interviewer prompt.

## Decisions & gotchas for future work

- Transcript stores only real conversation; `FIRST_MESSAGE_TRIGGER` is replayed
  as the first user turn at inference time so roles alternate (see
  `routes.ts` /interviews/:id/messages).
- Voice STT/TTS is browser-native — free, but Chrome-family only; Firefox falls
  back to text mode (`voiceSupported()` gate).
- `skills/` is resolved relative to `server/dist` at runtime; keep it at repo root.
- Server smoke-test data was cleaned from `~/.senior-bro` after verification.
